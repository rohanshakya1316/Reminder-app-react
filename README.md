# ⏰ Reminder App (React + Vite)

A simple and modern **Reminder Application** built using **React.js** and **Vite**.  
This app helps users create, manage, and delete reminders efficiently with a clean and responsive UI.

---

## 🚀 Features

- ➕ Add new reminders with title & details
- 📝 Edit existing reminders
- 🗑️ Delete reminders
- ✔️ Mark reminders as completed
- 📅 View all active reminders
- ⚡ Fast performance using Vite
- 📱 Fully responsive design

---

## 🛠️ Tech Stack

- React.js
- Vite
- Tailwind CSS
- JavaScript (ES6+)

---

## ⚙️ Getting Started

Follow these steps to run the project locally:

### 1. Clone the repository

```bash
git clone https://github.com/rohanshakya1316/Reminder-app-react.git
```

### 2. Navigate to project folder

```bash
cd Reminder-app-react
```

### 3. Install dependencies

```bash
npm install
```

### 4. Start development server

```bash
npm run dev
```

### 5. Open in browser

```
http://localhost:5173
```
